## Статус: Старт

Собираем веб-страницу «планировщик» для системы задач.

- Бэкенд: Google Apps Script с JSON API (готов)
- Фронтенд: в процессе планирования

## Ожидаем

- Детали API (эндпоинты, структура данных)
- Макет страницы
